Angular Data Binding

Visual summary of the 4 core binding types.

Interpolation

Template reads component value:

{{ '{{ interpolation }}' }}

Result: {{ interpolation }}

Property Binding

Component value sets DOM property:

[value]="propertyBinding"

Event Binding

DOM event calls component method:

(click)="increment()"

Two-way Binding

Input and component stay in sync:

[(ngModel)]="twoWayBinding"
