Angular Forms

Two approaches compared: Reactive Forms and Template-Driven Forms.

Reactive Form

FormBuilder

The form is defined in the TypeScript component using FormBuilder. Full control over validation and state.

@if (form.get('name')?.invalid && form.get('name')?.touched) { Mandatory name (min 2 characters) }
@if (reactiveSubmitted) {

{{ reactiveSubmitted }}

}

Template-Driven Form

ngModel

The form is defined in the HTML template using ngModel. Simpler, less control.

@if (nameCtrl.invalid && nameCtrl.touched) { Mandatory name (min 2 characters) }
@if (templateSubmitted) {

{{ templateSubmitted }}

}
