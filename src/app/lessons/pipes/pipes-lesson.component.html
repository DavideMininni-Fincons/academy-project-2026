Pipes

Transforms values in the template without modifying the original data.

Original value

Initial property of the component:

beerName = '{{ beerName }}'

{{ beerName }}

Text Pipes

Transforms strings:

uppercase | lowercase | capitalize

Uppercase: {{ beerName | uppercase }}

Lowercase: {{ beerName | lowercase }}

Capitalize: {{ beerName | capitalize }}

Formatting Pipes

Formats numbers and dates:

currency | date

Currency: {{ price | currency: 'EUR' : 'symbol' }}

Date: {{ today | date: 'short' }}

Json Pipe

Useful for quick debugging in the template:

{{ '{{ beer | json }}' }} 
{{ beer | json }}
