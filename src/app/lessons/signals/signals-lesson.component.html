Angular Signals

Modern reactive state management with signals, computed values, and effects.

signal()

Create reactive state:

count = signal(0)

Current value: {{ count() }}

computed()

Derive values from signals:

doubleCount = computed()

Original: {{ count() }}
Double: {{ doubleCount() }}

effect()

Trigger side effects on changes:

effect(() => {{ '{' }} ... {{ '}' }})
