# AcademyProject2026

A simple vanilla JavaScript project that displays "Hello Academy!" in the browser.

## Project Structure

```
academy-project-2026/
├── public/
│   └── favicon.ico      # Site icon
├── src/
│   ├── index.html       # HTML entry point
│   ├── script.js        # JavaScript code
│   └── styles.css       # CSS styling
└── README.md
```

## Setup

Open `src/index.html` in your browser.

## How It Works

- `index.html` contains the page structure and links to CSS and JS
- `script.js` dynamically updates the DOM when the page loads
- `styles.css` provides modern, responsive styling with CSS variables
